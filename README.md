# @thanh-libs/utils

Shared utility functions and hooks for System Core UI.

## Installation

```bash
yarn add @thanh-libs/utils
```

## Functions

- `pxToRem(px, htmlFontSize?)` — Convert pixel values to rem units
- `alpha(color, opacity)` — Apply alpha transparency to a color
